# 人生备忘录｜Life Orbit 生命轨道系统

- **用结构代替焦虑，用视觉代替混乱。**
 
## 1. 问题与三大痛点（Problem & Pain）

- **路演**  
  当我们的思维陷入混乱，确定的规则能恢复秩序。很多年轻人当下处于一个前所未有的“迷茫期”：信息过载、选择过剩，大脑每天被迫处理成千上万个碎片化念头，思维长期处在高熵混沌之中。传统备忘录和线性 To-Do 清单，只会把这些高维想法撕碎成一个个分散的线性片段，散落在各个应用和聊天窗口里；而通用 AI 又受限于上下文窗口，只能做记不住你的“聊天搭子”，既无法形成统一的人生视图，也无法成为我们大脑的下一代载体。  
  人生备忘录的使命，是让人用结构代替焦虑，用视觉代替混乱：用「生命光环」这套可视化轨道，把当下最紧要的事、中期成长与长期愿景放回各自的轨道；用本地向量数据库打造「外部海马体」，托管所有琐事和灵感，由 AI 负责长期记忆与主动提醒——在当下提醒你该做什么，在未来合适的时机唤回你真正重要的目标，让大脑不再时时刻刻担心“会不会忘了什么”。  
  正如古希腊法治取代人治，将理性引入社会；人生备忘录试图用一套可视化的理性结构，去取代原本脆弱的情绪本能：在今天，它是 2D 的人生光环系统；在 AR / VR 时代，这些光环会被拉伸成一枚时间光锥，让你站在光锥底部，一眼看清自己从哪里来、身在何处、将往何方。这不仅仅是一个工具，而是一场对现代人认知秩序的重构——它就是生命轨道，像空气一样支撑你的生活，照亮当下，也指引未来。

- **1.1 痛点一：脑海的“熵增”与念头的闪烁 (Mental Noise)**  
  **念头的随机闪烁**  
  我们脑海里每时每刻都闪烁着无数随机、跳跃的念头：上一秒想着“要去健身”，下一秒冒出“想学烘焙”，紧接着是“记得买牙膏”，这些念头从生存琐事到突发灵感再到宏大理想，无序地、随机地在脑中此起彼伏。  
  **分散记录与碎片堆积**  
  面对这些杂乱念头，我们只能分散地记录到各个备忘录、便签或聊天窗口里，形成多个彼此割裂的信息碎片堆。  
  **复盘困难与失控焦虑**  
  在这种工具形态下，宏大的愿景会被琐碎的日常持续淹没，我们在一遍遍完成“小事”的过程中，逐渐忘记了最初为什么出发、真正想要的生活是什么。

- **1.2 痛点二：生活是立体的，工具却是线性的 (The Linear Trap)**  
  **立体生活 vs 二维工具**  
  现实中的生活和思维是立体的，但传统的 To-Do List、备忘录这类工具却是二维、线性的，它们逼迫我们把“今晚买菜（生存）”和“三年后拍电影（理想）”挤在同一个扁平列表里。  
  **愿景被琐事淹没**  
  在这种工具形态下，宏大的愿景会被琐碎的日常持续淹没，我们在一遍遍完成“小事”的过程中，逐渐忘记了最初为什么出发、真正想要的生活是什么。  
  **信息孤岛化**  
  传统备忘录、微信文件助手、Notion 等工具进一步将信息割裂成一个个相互孤立的“信息岛”，既难以形成统一的人生视图，也难以支撑连续多年的长期规划。

- **1.3 痛点三：大脑内存溢出，AI 患有“健忘症” (AI Amnesia)**  
  **AI 记不住你**  
  我们试图把一部分记忆与思考交给 AI 打理，但通用 AI（如 ChatGPT）受限于上下文记忆窗口 (Context Window Limit)，对话一旦持续变长，它就会逐渐“忘记”你一两个月前提到的那些重要灵感和背景。  
  **无法成为人生容器**  
  在这种限制之下，AI 只能充当一次性的“聊天搭子”或短期思路整理助手，而无法成为一个真正承载你人生记忆、持续跟踪你长期目标与人生轨道的可靠容器。

## 2. 核心解决方案：降维打击 (The Solution)

- **2.1 视觉革命：生命轨道 (Life Orbit)**  
  **视觉与空间重构**  
  我们利用空间计算（Spatial Computing）和轨道化视觉结构，重构「如何看见自己的人生」这件事。  
  **2D 同心圆光环**  
  在当下阶段，人生轨道以 2D 同心光环呈现：中心越近，代表越「当下、越紧要」；越向外扩展，时间尺度越长、越偏向长期方向感。  
  **可配置光环层级**  
  光环层级并非固定死板的三层结构，系统提供默认示例，但每一环的数量、名称与含义都可以由用户自由增减和重命名，适配不同的人生阶段与世界观。  
  **典型三层示例**  
  一种典型示例是：中心光环放置当下最紧要、必须推进的事项；中间光环承载中期的成长项目与沉淀；外层光环则安放长期愿景与人生方向，从而让「今晚必须写完简历」和「我要拍一部电影」不再挤在同一条列表里。  
  **光锥与时间纵深**  
  面向未来的 AR / VR 时代，人生光环会从二维平面「立起来」，演变为具有时间纵深的光锥 (The Light Cone)：戴上 AR 眼镜从侧面看，层层轨道被拉伸为时间锥，底部沉淀过去，中段是当下截面，顶端指向未来，侧面可以挂载阶段性记录与照片墙，一眼看清自己从哪里来、身在何处、将往何方。

- **2.2 记忆革命：外部海马体 (External Hippocampus)**  
  **本地无限记忆**  
  本地无限记忆：部署本地向量数据库 (Local Vector DB)，彻底解决 AI 遗忘问题，让机器端真正具备「长期记住你」的能力。  
  **彻底托管**  
  彻底托管：你只管把脑海中那些乱七八糟的念头、琐事、灵感和担忧全部“丢”进去，由 AI 加数据库组成的「外挂海马体」负责永久存储与结构化整理，无论跨度多久，AI 都能通过 RAG（检索增强）技术瞬间调取。  
  **主动提醒**  
  主动提醒：我们很多时候的焦虑，并不是事情本身，而是「怕自己忘了这件事」。人生备忘录通过对时间、场景和轨道层级的识别，自动推送当下最应该处理的事项，以及在合适的时机提醒你那些重要但不紧急的长期目标——用 AI 来记住「当下该做什么、未来该做什么」，而不是让大脑一直绷着这根弦。  
  **价值释放**  
  价值释放：当「记住所有事情」这件事从大脑迁移到外部海马体，大脑可以从时时刻刻操心、反复确认「我有没有忘记什么」的负担中解放出来，把有限的注意力集中在真正重要的创造、决策与体验上。

- **2.3 人机共创：掌控感 (Agency)**  
  **AI 预判**  
  AI 预判：系统基于 CBT（认知行为疗法）等心理学逻辑，对用户输入的念头进行初步识别与标注，例如自动将“想去大理”归类到愿景 / 体验层，将“买电费”“看牙”归类到生存维护层。  
  **手动掌控**  
  手动掌控：最终决定权永远在用户手里，你可以手动拖拽节点，重新排列光环层级与优先级，让 AI 提供建议，但不替你做主，确保人生的主动权始终掌握在自己手中。

## 3. 核心功能与体验 (The Experience)

- **功能总览**  
  [细节还原]：包含“漂浮入口”、“专注 UI”、“未来镜焚烧”、“随机社交”与“求职 API”。

- **3.1 空间入口与问询 (The Floating Space)**  
  **漂浮空间入口**  
  入口形态是一个漂浮空间，你的想法以「思维碎片光点」的方式出现（例如：减肥、搬家、做视频、报一个课程）。  
  **引导式问询**  
  抓取任意一个光点后，AI 通过简短的引导式提问（“这件事对你重要吗？”、“如果不做会怎样？”、“是什么让你停滞？”），帮助你澄清这条念头的真实层级与时间尺度，并安放到对应的人生光环之中。

- **3.2 专注引导 (Focus-Guided UI)**  
  **眼动追踪专注模式**  
  针对注意力涣散人群，系统利用眼动追踪技术，你看哪里，哪里的轨道节点就亮起，不看的地方自动虚化（Bokeh Effect），以最少的信息刺激，获得最大的专注。

- **3.3 Future Mirror（未来镜模块）**  
  **生成式愿景视频**  
  集成 Sora / Runway API，生成“达成目标后的未来自我”视频。  
  **焚烧期机制**  
  生成的未来影像仅可查看保留 3 分钟，之后不可逆消散且只保留缩略图。  
  **设计哲学**  
  设计哲学：未来只可一瞥，不可沉溺，用“失去的恐惧”倒逼当下的行动，防止用户沉迷于虚幻的想象。

- **3.4 社交与生态 (Ecosystem)**  
  **3.4.1 随机共鸣 (Orbital Resonance)**  
  **相似轨道匹配**  
  系统可随机匹配全球另一个拥有“相似轨道”的陌生人，在 AR 空间中看到对方的“人生光环”与你交叠，你看到的不是照片，而是他正在经历的挑战和梦想。  
  **隐私控制与社区愿景**  
  隐私开关允许用户选择是否开启，逐步形成基于深度成长共鸣的高价值在线社区。  
  **3.4.2 职业轨道 (Career API)**  
  **现实职场连接**  
  接入 LinkedIn / Boss 直聘 API，当“成长层”技能节点点亮时，系统自动依据轨道优先级推荐现实世界的岗位路径，打通从“成长”到“变现”的闭环。

## 4. 蓝海市场在哪里 (The Market)

- **4.1 硬件爆发 vs 软件荒漠**  
  **硬件就绪**  
  随着 Apple Vision Pro、Ray-Ban Meta、Rokid、夸克 AI 眼镜、理想 AI 眼镜等 AR 眼镜的层出不穷，空间计算设备正在从极客玩具走向大众日常，硬件的普及已成定局。  
  **软件缺位**  
  目前的 AR 应用市场充斥着看电影、切水果等娱乐应用，但在“个人效率与认知管理”领域，仍是一片广阔的蓝海。当人们戴上眼镜，需要的不是另一个悬浮在空中的 2D 窗口，而是一个原生的 3D 人生操作系统，Life Orbit 抢占的正是这一生态位。

- **4.2 全民刚需 (Mass Market)**  
  **记录的刚性需求**  
  “记录”是人类最底层的需求之一，就像每个人都需要穿鞋一样，每个人都需要一个工具来管理自己的人生。  
  **代际升级路径**  
  从纸笔到 Notion，再到空间计算时代的人生备忘录，只要你有大脑，只要你有那些闪烁不定的念头，你就需要这个“外部海马体”。  
  **产品定位**  
  这不是小众极客的玩具，而是下一代通用的数字生活基础设施。

## 5. 快速开始 (Run Locally)

**环境要求**: Node.js v15+

1. **安装依赖 (Install)**:
   ```bash
   npm install
   ```

2. **配置环境 (.env.local)**:
   在项目根目录创建 `.env.local` 文件，并填入你的 Gemini API Key：
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```
   *(Life Orbit 使用 Gemini 2.0 Flash 模型进行 RAG 推理与向量计算)*

3. **启动项目 (Start)**:
   ```bash
   npm run dev
   ```
   
打开浏览器访问 `http://localhost:3000` 即可体验。
